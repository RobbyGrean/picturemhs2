คู่มือการใช้งาน PICMHS2

จัดเก็บและค้นหารูปกิจกรรมอย่างเป็นระบบ

คู่มือนี้อธิบายการค้นหารูป การสร้างและอัปโหลดกิจกรรม การแก้ไขหรือย้าย การเผยแพร่ และการลบแบบเก็บไฟล์ไว้ใน Drive

สารบัญ ค้นหารูปสำหรับบุคคลทั่วไป เข้าสู่ระบบสำหรับผู้อัปโหลด สร้างกิจกรรมใหม่ อัปโหลดเพิ่ม แก้ไขหรือย้ายกิจกรรม การเผยแพร่ ลบ และกู้คืน จัดการผู้อัปโหลดสำหรับผู้ดูแลระบบ กติกาการจัดการไฟล์โดยตรง แก้ปัญหาเบื้องต้น

ค้นหารูปสำหรับบุคคลทั่วไป

ไม่ต้อง Google Sign-In และไม่มีสิทธิ์อัปโหลดหรือแก้ไขข้อมูล

  1. เปิดหน้า ค้นหารูปสาธารณะ
  2. ดูกิจกรรมล่าสุดที่เผยแพร่ได้ทันที สูงสุด 10 รายการ
  3. เมื่อต้องการค้นหา ให้เลือกปี พ.ศ. ก่อน ส่วนเดือน วัน หมวด และชื่อกิจกรรมเป็นตัวเลือก
  4. พิมพ์ชื่อกิจกรรมเต็มหรือคำบางส่วน เช่น “ประชุม”
  5. กด ค้นหา และเลือกกิจกรรมจากผลลัพธ์
  6. กด เปิดโฟลเดอร์รูป เพื่อดูหรือดาวน์โหลดผ่าน Google Drive
ระบบแสดงเฉพาะกิจกรรม active ที่ตั้งเป็น public จำนวนไฟล์เป็นค่าประมาณจาก metadata ไม่ใช่การนับไฟล์สดจาก Google Drive

เข้าสู่ระบบสำหรับผู้อัปโหลด

  1. เปิด เว็บไซต์หลัก
  2. กด Google Sign-In และเลือกบัญชีที่ได้รับอนุญาต
  3. อนุญาตสิทธิ์ Google Drive เมื่อระบบถาม
  4. เมื่อผ่านรายชื่อผู้ได้รับอนุญาตและสิทธิ์ Google Drive ระบบจะแสดงหน้าจัดการกิจกรรม
ผู้ที่มีหน้าที่ค้นหาอย่างเดียวไม่ควรได้รับสิทธิ์ Editor ให้ใช้หน้าค้นหารูปสาธารณะแทน

สร้างกิจกรรมใหม่

  1. เลือกหมวด ปี พ.ศ. เดือน และวัน ปีที่ใช้ได้คือปีปัจจุบันและย้อนหลังสองปีตามเวลา Bangkok
  2. เลือกชื่อสำเร็จรูปหรือพิมพ์ชื่อกิจกรรมเอง
  3. เลือกการเผยแพร่ ค่าเริ่มต้นคือ เผยแพร่และบุคคลทั่วไปค้นหาได้
  4. เลือกรูปหรือวิดีโอและตรวจรายการไฟล์
  5. กด อัปโหลดกิจกรรมใหม่
  6. รอจนสร้างโฟลเดอร์ อัปโหลดไฟล์ และบันทึกข้อมูลกิจกรรมสำเร็จครบทุกขั้น
อย่าปิดหน้าเว็บระหว่างอัปโหลด หากหมวด วันที่ และชื่อซ้ำกับกิจกรรม active ระบบจะไม่สร้างรายการใหม่ ให้ค้นหาแล้วใช้อัปโหลดเพิ่ม

ค้นหาและอัปโหลดเพิ่ม

  1. เปิดแท็บค้นหาในระบบผู้อัปโหลด
  2. เลือกปี พ.ศ. ก่อนค้นหา ส่วนตัวกรองอื่นเป็นตัวเลือก
  3. ค้นหาและเลือกกิจกรรมเดิม
  4. เลือกไฟล์ที่ต้องการเพิ่ม
  5. กด อัปโหลดเพิ่ม และรอจนข้อมูลกิจกรรมอัปเดต

ผู้ใช้ที่ได้รับอนุญาตอัปโหลดเพิ่มในกิจกรรม active ที่อยู่ในช่วงสามปีได้ แต่การอัปโหลดเพิ่มไม่เปลี่ยนเจ้าของกิจกรรม

กิจกรรมเก่ากว่าช่วงสามปีค้นหาและเปิด Drive ได้ แต่จะแสดงป้ายอ่านอย่างเดียวและไม่มีปุ่มอัปโหลดเพิ่ม

แก้ไขหรือย้ายกิจกรรม

เจ้าของกิจกรรมและผู้ดูแลระบบเท่านั้นที่จัดการกิจกรรม active ภายในช่วงปีที่แก้ไขได้

  1. เปิดแท็บ ค้นหา / อัปโหลดเพิ่ม เลือกปี แล้วค้นหากิจกรรม
  2. กด แก้ไข/ย้ายกิจกรรม
  3. แก้หมวด ปี เดือน วัน ชื่อกิจกรรม หรือการเผยแพร่
  4. กด บันทึกการแก้ไข
  5. ปิดหน้าต่างด้วยปุ่ม ยกเลิก, ปุ่ม × หรือปุ่ม Escape บนแป้นพิมพ์
ระบบย้ายหรือเปลี่ยนชื่อโฟลเดอร์เดิม จึงรักษา folder ID, URL, เจ้าของ และไฟล์ทั้งหมดไว้ หากปลายทางมีชื่อซ้ำ ระบบเติม (1), (2) โดยไม่รวมโฟลเดอร์

การเผยแพร่ ลบ และกู้คืน

การเผยแพร่

เผยแพร่และบุคคลทั่วไปค้นหาได้ จะแสดงใน Public Search และกิจกรรมล่าสุด ส่วน ไม่เผยแพร่ จะแสดงเฉพาะผู้ใช้ที่ล็อกอิน

ไม่เผยแพร่เป็นแบบ unlisted ไม่ใช่ Drive private ผู้ที่มีลิงก์ Drive เดิมอาจยังเปิดได้ตาม permission

ลบกิจกรรม

  1. กด ลบกิจกรรมนี้ จากผลค้นหาหรือพื้นที่อันตรายในหน้าต่างแก้ไข
  2. อ่านข้อความยืนยันแล้วกดยืนยันเฉพาะเมื่อต้องการซ่อนรายการจากระบบค้นหา
  3. ระบบเปลี่ยนรายการเป็น archived แต่ไม่ลบโฟลเดอร์หรือไฟล์ใน Google Drive

Admin สามารถเลือก แสดงรายการที่ลบแล้วสำหรับกู้คืน ในหน้าค้นหา แล้วกด กู้คืนกิจกรรม

จัดการผู้อัปโหลดสำหรับผู้ดูแลระบบ

เมนูนี้แสดงเฉพาะผู้ดูแลที่กำหนดไว้ในระบบหลังบ้าน ผู้ที่เพิ่มใหม่จะเป็นผู้อัปโหลดและไม่มีสิทธิ์เพิ่มผู้ใช้อื่น

  1. เปิดแท็บ จัดการผู้อัปโหลด
  2. กรอกอีเมล ชื่อที่แสดง และหมายเหตุ เช่น ฝ่ายหรือหน้าที่
  3. กด บันทึกผู้อัปโหลด
  4. เพิ่มอีเมลเดียวกันเป็น Editor บน Drive root ด้วยตนเอง
  5. ใช้ ปิดสิทธิ์ เพื่อระงับการใช้งานโดยไม่ลบข้อมูล
  6. ใช้ เปิดสิทธิ์ เมื่อต้องการให้กลับมาใช้งาน
ช่องหมายเหตุเป็นคำอธิบาย ไม่ใช่บทบาทของระบบ ผู้ดูแลถูกกำหนดจากระบบหลังบ้านเท่านั้น และระบบจะไม่เพิ่มหรือลบสิทธิ์ Editor ใน Google Drive อัตโนมัติ

กติกาการจัดการไฟล์โดยตรง

ระบบค้นหาจากข้อมูลกิจกรรมใน Google ชีต และไม่สแกน Google Drive อัตโนมัติ

การกระทำใน Driveผลในระบบ
เพิ่มไฟล์ในกิจกรรมเดิมเปิด Drive แล้วเห็นไฟล์ แต่จำนวนไฟล์อาจไม่เพิ่ม
สร้างโฟลเดอร์กิจกรรมเองค้นหาไม่เจอ เพราะไม่มีข้อมูลกิจกรรมในระบบ
เปลี่ยนชื่อหรือย้ายโฟลเดอร์อย่าทำใน Drive ให้ใช้ปุ่มแก้ไข/ย้ายกิจกรรมเพื่ออัปเดต Drive และ metadata พร้อมกัน
ลบโฟลเดอร์อย่าทำใน Drive ให้ใช้ลบกิจกรรมนี้ ซึ่งซ่อนผลค้นหาแต่เก็บไฟล์ไว้
กิจกรรมที่ต้องการให้ระบบดูแลต้องสร้าง เพิ่มไฟล์ แก้ไข ย้าย และลบผ่าน PICMHS2 การจัดการโดยตรงใน Drive ถือว่าอยู่นอกการรับประกันของระบบ

ภาพตัวอย่างกิจกรรม

กิจกรรมที่อัปโหลดผ่านระบบหลังอัปเดตนี้ จะใช้ภาพแรกเป็น thumbnail บนผลค้นหา หากไม่มีภาพระบบจะลองใช้ภาพตัวอย่างของวิดีโอ และถ้า Drive ยังประมวลผลไม่ได้หรือผู้ชมไม่มีสิทธิ์ ระบบจะแสดงภาพประกอบหมวดแทน

กิจกรรมเก่าต้องให้ผู้ดูแลรัน backfillActivityCovers ใน Apps Script ก่อน ภาพตัวอย่างจึงจะปรากฏ การค้นหาปกติไม่สแกนโฟลเดอร์ Drive จึงไม่เพิ่มภาระ API ต่อการ์ด

แก้ปัญหาเบื้องต้น

อาการแนวทางตรวจสอบ
ค้นหาไม่พบยืนยันว่าเลือกปีแล้ว ลดตัวกรอง ตรวจชื่อกิจกรรม และตรวจว่ารายการยัง active
ไม่พบในเว็บสาธารณะตรวจว่า visibility เป็น public และ activity status เป็น active
ไม่มีปุ่มแก้ไขหรือลบต้องเป็นเจ้าของกิจกรรมหรือ Admin และกิจกรรมต้องอยู่ในช่วงปีปัจจุบันย้อนหลังสองปี
กิจกรรมเป็นอ่านอย่างเดียวกิจกรรมเก่ากว่าช่วงสามปี จึงเปิด Drive ได้แต่แก้ไขหรือลงไฟล์เพิ่มไม่ได้
เปิดโฟลเดอร์ไม่ได้แจ้งผู้ดูแลตรวจการแชร์ Google Drive และข้อจำกัดการเข้าถึงของโฟลเดอร์ย่อย
เข้าสู่ระบบไม่ได้ตรวจว่าใช้อีเมลที่อยู่ในรายชื่อผู้ได้รับอนุญาต และได้รับสิทธิ์ Editor ใน Google Drive
อัปโหลดไม่ได้ลองเข้าสู่ระบบใหม่ ตรวจการเชื่อมต่ออินเทอร์เน็ต และอย่าปิดหน้าเว็บระหว่างทำงาน
จำนวนไฟล์ไม่ตรงมีการเพิ่มหรือลบไฟล์ผ่าน Drive โดยตรง ระบบไม่ sync อัตโนมัติ
ค้นหาช้าเป็นบางครั้งApps Script อาจเริ่มระบบใหม่ กรุณารอสักครู่แล้วลองอีกครั้ง
