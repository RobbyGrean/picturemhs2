PICMHS2 USER GUIDE

จัดเก็บและค้นหารูปกิจกรรมอย่างเป็นระบบ

คู่มือนี้อธิบายการค้นหารูป การสร้างกิจกรรมใหม่ การอัปโหลดเพิ่ม และขอบเขตของข้อมูลที่ระบบดูแล

สารบัญ ค้นหารูปสำหรับบุคคลทั่วไป เข้าสู่ระบบ uploader สร้างกิจกรรมใหม่ อัปโหลดเพิ่ม กติกา manual upload แก้ปัญหาเบื้องต้น

ค้นหารูปสำหรับบุคคลทั่วไป

ไม่ต้อง Google Sign-In และไม่มีสิทธิ์อัปโหลดหรือแก้ไขข้อมูล

  1. เปิดหน้า ค้นหารูปสาธารณะ
  2. เลือกปี เดือน วัน หรือหมวดเท่าที่ทราบ ตัวกรองทุกช่องเป็นตัวเลือก
  3. พิมพ์ชื่อกิจกรรมเต็มหรือคำบางส่วน เช่น “ประชุม”
  4. กด ค้นหา และเลือกกิจกรรมจากผลลัพธ์
  5. กด เปิดโฟลเดอร์รูป เพื่อดูหรือดาวน์โหลดผ่าน Google Drive
จำนวนไฟล์ที่แสดงเป็นจำนวนโดยประมาณจาก metadata ไม่ใช่การนับไฟล์สดจาก Drive

เข้าสู่ระบบ uploader

  1. เปิด เว็บไซต์หลัก
  2. กด Google Sign-In และเลือกบัญชีที่ได้รับอนุญาต
  3. อนุญาตสิทธิ์ Google Drive เมื่อระบบถาม
  4. เมื่อผ่าน allowlist และ Drive permission ระบบจะแสดงหน้าจัดการกิจกรรม
ผู้ที่มีหน้าที่ค้นหาอย่างเดียวไม่ควรได้รับ Editor permission ให้ใช้หน้า Public Search แทน

สร้างกิจกรรมใหม่

  1. เลือกหมวด ปี พ.ศ. เดือน และวัน
  2. เลือกชื่อสำเร็จรูปหรือพิมพ์ชื่อกิจกรรมเอง
  3. เลือกรูปหรือวิดีโอและตรวจรายการไฟล์
  4. กด อัปโหลดกิจกรรมใหม่
  5. รอจนสร้างโฟลเดอร์ อัปโหลดไฟล์ และบันทึก metadata สำเร็จครบทุกขั้น
อย่าปิด browser ระหว่างอัปโหลด และอย่าสร้างโฟลเดอร์กิจกรรมซ้ำเองใน Drive

ค้นหาและอัปโหลดเพิ่ม

  1. เปิดแท็บค้นหาในระบบ uploader
  2. ค้นหาและเลือกกิจกรรมเดิม
  3. เลือกไฟล์ที่ต้องการเพิ่ม
  4. กด อัปโหลดเพิ่ม และรอจน metadata อัปเดต

การใช้ flow นี้ทำให้จำนวนไฟล์และเวลาแก้ไขล่าสุดอัปเดตตามระบบ

กติกา manual upload

ระบบค้นหาจาก Google Sheets metadata และไม่สแกน Drive อัตโนมัติ

การกระทำใน Driveผลในระบบ
เพิ่มไฟล์ในกิจกรรมเดิมเปิด Drive แล้วเห็นไฟล์ แต่จำนวนไฟล์อาจไม่เพิ่ม
สร้างโฟลเดอร์กิจกรรมเองค้นหาไม่เจอ เพราะไม่มี metadata
เปลี่ยนชื่อหรือย้ายโฟลเดอร์ชื่อ หมวด หรือ path ในผลค้นหาอาจไม่ตรง
ลบโฟลเดอร์ผลค้นหาอาจยังอยู่ แต่ลิงก์เปิดไม่ได้
กิจกรรมที่ต้องการให้ระบบดูแลต้องสร้างและเพิ่มไฟล์ผ่าน PICMHS2 ข้อมูล manual ถือว่าอยู่นอกการรับประกันของระบบ

แก้ปัญหาเบื้องต้น

อาการแนวทางตรวจสอบ
ค้นหาไม่พบลดตัวกรอง ตรวจชื่อกิจกรรม และยืนยันว่ากิจกรรมสร้างผ่านระบบ
เปิดโฟลเดอร์ไม่ได้แจ้ง admin ตรวจ Drive sharing และ child limited access
Login ไม่ได้ตรวจว่าใช้อีเมลที่อยู่ใน allowlist และได้รับ Drive Editor permission
อัปโหลดไม่ได้ลอง Login ใหม่ ตรวจ internet และอย่าปิด browser ระหว่างทำงาน
จำนวนไฟล์ไม่ตรงมีการเพิ่มหรือลบไฟล์ผ่าน Drive โดยตรง ระบบไม่ sync อัตโนมัติ
ค้นหาช้าเป็นบางครั้งApps Script อาจเริ่มระบบใหม่ กรุณารอสักครู่แล้วลองอีกครั้ง
